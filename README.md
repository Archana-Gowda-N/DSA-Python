# Python Basics – DSA Practice

This repository contains beginner Python programs for practicing basic programming concepts.

## Programs Included

1. **swap.py**

   * Swaps two numbers.

2. **evenodd.py**

   * Checks whether a number is even or odd.

3. **posneg.py**

   * Checks whether a number is positive, negative, or zero.
